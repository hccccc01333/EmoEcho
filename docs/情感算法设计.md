# 情感算法设计：E3-Score v0.2（心理学增强 + 自学习）

> 目标：把"情感陪伴"从文案概念变成可执行、可评估、可迭代的算法系统。

---

## 一、心理学理论基础

本算法融合 5 套经典心理学理论，每套理论对应具体的算法变量和策略：

| 理论 | 核心思想 | 映射到的算法变量 | 驱动的策略 |
|---|---|---|---|
| 依恋理论 (Bowlby) | 人的依恋风格影响关系中的情绪反应模式 | `attachmentSignal` | 回复距离感（贴近 vs 留白） |
| 情绪调节过程模型 (Gross) | 情绪调节分阶段：触发前/爆发中/恢复期 | `emotionStage` | 阶段性策略（重评/转移/呼吸引导） |
| 非暴力沟通 NVC (Rosenberg) | 观察-感受-需要-请求四步框架 | `nvcCompleteness` | 输出结构约束（减少评判性语言） |
| 动机式访谈 MI (OARS) | 用开放提问和反映倾听引导改变 | `changeReadiness` | 当用户求改变时少说教多引导 |
| 认知行为 CBT | 识别认知扭曲并温和重构 | `cognitiveDistortionScore` | 高分时触发"温和重构"而非硬反驳 |

## 二、E3-Score 三维评分

### Empathy（共情需求强度）

```text
Empathy = 0.35 * emotionIntensity     ← 情绪词密度
        + 0.25 * distressSignal       ← NVC 中"感受+需要"信号
        + 0.20 * memoryPainWeight     ← 记忆中高痛苦事件命中
        + 0.10 * nightBoost           ← 深夜权重 (23:00-05:00)
        + 0.10 * recentConflictRate   ← 近期冲突密度
```

### Stability（对话稳定度）

```text
Stability = 1 - (
    0.40 * conflictProbability        ← 冲突词命中
  + 0.35 * cognitiveDistortionRate   ← CBT 认知扭曲匹配数
  + 0.25 * topicDriftRate            ← 话题跳跃频率
)
```

### Boundary（边界风险度）

```text
Boundary = 0.50 * riskKeywordScore    ← 自伤/暴力关键词
         + 0.30 * coercionIntent      ← 胁迫/控制意图
         + 0.20 * dependencyRisk      ← 过度依赖信号
```

所有分数归一化到 [0, 1]，每轮写入会话状态缓存，支持 EMA 平滑。

---

## 三、策略路由决策树

```text
if Boundary >= 0.65:
    -> SafeGuardPolicy（安全降级 + 求助提示 + 限制深度模拟）

elif Empathy >= 0.70 and Stability >= 0.45:
    -> ComfortPolicy（NVC 框架回应 + 高共情 + 温和引导）

elif Stability < 0.45:
    -> StabilizePolicy（Gross 模型降冲突 + 不接招攻击 + 冷静回应）

elif changeReadiness detected:
    -> GuideChangePolicy（MI-OARS 技巧 + 开放提问 + 反映倾听）

else:
    -> DefaultPersonaPolicy（正常人格表达）
```

## 四、自学习机制（三层）

### L1 会话级（秒级生效）

- 检测隐式偏好信号（"好真实" / "太假了"）
- EMA 更新当前会话的语气/共情/长度参数
- 仅影响当前会话，不改核心人格

### L2 纠正记忆（持久生效）

- 检测显式纠正模式（"ta不会这么说" / "别这样说" / "记错了"）
- 写入 `correction_memory` 表，高优先级
- 每次生成前先检索纠正项，覆盖旧行为

### L3 周期性微调（慢速）

- 每 50 轮做一次"偏好回放"
- 将稳定偏好合并进 `adaptive_profile`
- 只更新"可学习字段"（语气、常用词、安抚方式）
- 不更新"边界字段"（安全规则、合规策略）

### 核心原则

**学习的是"表达策略"，不学习"价值边界"。**

```text
new_weight = old_weight * (1 - alpha) + feedback_signal * alpha
alpha = 0.1（默认 EMA 系数）
```

## 五、可评估指标

| 指标 | 定义 | 目标 |
|---|---|---|
| E3路由准确率 | 策略选择与人工标注一致率 | >= 80% |
| 风险拦截召回率 | 高风险输入被成功拦截比例 | >= 95% |
| 情绪安抚成功率 | 负向情绪在 3 轮内情绪指标下降比例 | >= 60% |
| 边界稳定率 | 连续追问下仍保持边界的比例 | >= 90% |
| 人格一致性 | 30 题标准问答集一致率 | >= 75% |
| 纠正响应率 | 用户纠正后下次输出改变比例 | >= 85% |
