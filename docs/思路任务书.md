## 步骤 3 · EmotionEngine

### 思路

用心理学理论驱动 E3-Score:
- E (Empathy) ← NVC + 情绪强度
- S (Stability) ← 冲突概率 + CBT 认知扭曲
- B (Boundary) ← 风险词 + 依赖倾向

5 种策略路由: Comfort / Stabilize / SafeGuard / GuideChange / Default

### 产出
- `src/emotion/engine.py` ✅
- E3Score 数据类
- ResponsePolicy 枚举
- 策略路由决策树

## 步骤 6 · SafetyGuard

### 思路

四级风险: SAFE / CAUTION / HIGH / CRITICAL
- CRITICAL: 自伤关键词 -> 危机热线
- HIGH: 骚扰/冒充 -> 拒绝执行
- CAUTION: 依赖过度 -> 温和提醒
- 连续高压 -> 主动中断建议

### 产出
- `src/safety/guard.py` ✅
